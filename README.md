# 🧪 Machine Learning Laboratory Programs

A React-Vite dashboard displaying practical implementations and outputs of core Machine Learning laboratory tasks.

## 🚀 Features
- **Code Viewer**: Live preview of ML algorithms (KNN, Naive Bayes, Decision Trees, K-Means).
- **Output Plots**: Interactive graphs plotting model decision boundaries.

## 🛠️ Tech Stack
- React 19, Vite 8, React Router
